## Project Description

This website is a personal homepage for Akhilan Anbu, a Computer Science graduate student interested in AI/ML, web development, research, and music. The homepage is designed to give visitors a quick understanding of who I am, what I work on, and what skills I bring.

The website includes a home page, a projects page, and a creative AI-generated page. The design focuses on simple navigation, meaningful content, responsive layout, organized files, and accessibility.

## Creative Addition

The projects page includes an original JavaScript-based project filter. Users can click category buttons such as:

- All
- AI/ML
- Research
- Web Dev
- Music

When a button is clicked, the project cards update dynamically based on the selected category. This feature was written using ES6 JavaScript modules without using jQuery or external component libraries.

## Instructions to Build and Run

1. Clone or download this repository.
2. Open the project folder in VS Code.
3. Install dependencies:

npm install

4. Format the project using Prettier:

npm run format

5. Run ESLint:

npm run lint

5. Run ESLint:
   npm run lint

6. Open `index.html` in a browser, or use the VS Code Live Server extension.
