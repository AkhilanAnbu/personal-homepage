# Design Document: Personal Homepage

## Project Description

This project is a personal homepage for Akhilan Anbu. The goal is to present my background, skills, projects, research interests, and creative identity in a clean and responsive front-end website. The website is built using vanilla HTML5, CSS3, and ES6 JavaScript modules.

## User Personas

### Persona 1: Recruiter

A recruiter wants to quickly understand my technical skills, project experience, and contact information.

### Persona 2: Professor or Researcher

A professor wants to see my research interests, AI/ML background, and academic project experience.

### Persona 3: Classmate or Peer

A classmate wants to learn more about me, explore my projects, and understand what kind of work I enjoy building.

## User Stories

1. As a recruiter, I want to quickly view Akhilan's skills and projects so that I can understand his fit for technical roles.
2. As a professor, I want to see Akhilan's AI and research interests so that I can understand his academic background.
3. As a visitor, I want to navigate between pages easily so that I can explore the homepage without confusion.
4. As a user, I want to filter projects by category so that I can quickly find the type of work I care about.
